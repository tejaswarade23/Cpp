## 🚀 Getting Started

### 🔧 Prerequisites

* C++ Compiler (GCC / g++)
* Code Editor (VS Code, CodeBlocks, etc.)

---

### ▶️ How to Run a Program

1. Clone the repository:

   ```bash
   git clone https://github.com/tejaswarade23/Cpp.git
   ```

2. Navigate to the folder:

   ```bash
   cd Cpp
   ```

3. Compile the program:

   ```bash
   g++ filename.cpp -o output
   ```

4. Run the program:

   ```bash
   ./output
   ```
